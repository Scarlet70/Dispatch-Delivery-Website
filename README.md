# ETransit Dispatch Website

ETransit is a modern and responsive delivery service website built with **React.js**. The platform simulates a real-world dispatch agency that allows users to place delivery orders, track shipments, explore pricing packages, and apply to become delivery riders.

---


---

## Project Overview

This website represents the digital presence of a logistics and dispatch company. The project focuses on clean UI/UX design, responsive layouts, and engaging animations to create a professional delivery service experience.

The website provides users with essential service information and interactive sections that improve usability and customer engagement.

---

## Features

### Customer Features
- Delivery service overview
- Shipment tracking
- Pricing packages
- Contact and support section

### Rider Recruitment
- Become a rider section
- Rider application workflow
- Rider benefits and requirements

### UI/UX Features
- Fully responsive design
- Animated SVG illustrations
- Smooth navigation highlighting
- Interactive pricing cards
- Animated hero section
- Hover and transition effects
